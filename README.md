# Medi_Guide
MediGuide is an AI-powered healthcare web application that provides medicine recommendations based on symptoms, WhatsApp medicine reminders, BPM monitoring, voice-based symptom input, GPS-based hospital finding, and secure user management using Flask, NLP, Selenium, and APScheduler.
MediGuide – Medical Guidance and Reminder System

MediGuide is a smart healthcare web application designed to assist users with medicine recommendations, medication management, and basic health monitoring. The system combines Artificial Intelligence, Machine Learning, Automation, and Healthcare Assistance into a single platform.

Features
AI-Based Medicine Recommendation
Voice-Based Symptom Input
WhatsApp Medicine Reminder Automation
BPM (Heart Rate) Monitoring
GPS-Based Nearby Hospital Finder
User Authentication & Profile Management
Admin Dashboard
Multi-Language Support
Background Reminder Scheduling

Project Workflow
User enters symptoms via text or voice.
NLP-based recommendation engine analyzes symptoms.
System suggests medicines and healthcare advice.
Users can schedule medicine reminders.
APScheduler monitors reminder timings.
Selenium automates WhatsApp reminder notifications.
BPM module provides heart rate monitoring.
GPS module helps locate nearby hospitals.
Objective

To provide an intelligent, accessible, and user-friendly healthcare assistant that improves medication adherence, health monitoring, and healthcare accessibility.

Tech Stack: Python, Flask, JavaScript, HTML, CSS, NLP, Scikit-learn, Selenium, APScheduler, JSON, CSV, ngrok.
